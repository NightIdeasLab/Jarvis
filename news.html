News 02.10.2013

Beta Testing Signup has ended

Finally after almost one year Jarvis has been updated to version 0.4.2. For the last year I was busy with my master thesis and my hobby the robot NAO and RoboCup.

Enough about me, lets talk about Jarvis 0.4.2:

  • New - Displaying an image for today's weather.
  • New - Enabling or disabling the 5 day forecast.
  • New - Change of the temperature and time style.
  • Changed - Brand new Changelog window.
  • Changed - Dropped 32-bit support, the app now is only 64-bit.
  • Changed - Minimum operating system is 10.6 (Snow Leopard).
  • Fixed - Memory leak.
  • New - Reading the user name.
  • New - Menu Item with useful commands for Jarvis.
  • New - Preferences view for each element.
  • New - Only one instance of the app will run.
  • Fixed - Weather system.
  • New - Dock icon menu.
  • New - Clicking on the Jarvis icon from the main window will refresh the report.
  • New - Added a feedback panel.

If you find bugs or have suggestions please fell free to contact me by email or using the new feedback system in the Help menu of the application. Thank you.

More to come, stay tuned.

News 30.05.2012

Jarvis 0.4 just arrived with the following features and fixes:

  • The update feed URL.
  • Gathering anonymous statistics.
  • Donate panel.
  • Update Preference panel.
  • Support for Xcode 4.3.
  • Incorrect display of the minutes in the hour if they where from 1 to 10.
  • The company name from buHuHu to Night Ideas Lab.
  • Added Contributors button and Contributors panel with contact buttons for me and the original developer.
  • Full-Screen Support.
  • Removed the "About Jarvis" title name from the About panel to be compliant with Mac OS X Human Interface Guidelines.
  • The ChangeLog menu now shows a local html page.
  • Preferences panel.
  • Version and Build number in the About panel.
  • Support for 10.7 (Lion) and Xcode 4.2.1.
  • License panel and license to GNU GENERAL PUBLIC LICENSE version 3.
  • Minimum Operating System is 10.5 (Leopard).
