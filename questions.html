Questions & Answers

What is Jarvis

Jarvis is a digital life assistant that assists with your daily life by acting as an alarm clock, informing you of the latest news headlines and the forecast of the weather for the next 5 days amongst other things. For the moment the application can do:

  • Greetings.
  • Time and date.
  • Calendar events.
  • Reminders (from iCal).
  • Current day weather conditions (from OpenWeatherMap).
  • Unread email count and VIP email count (from Apple Mail app).
  • News headlines (from NYTimes).
  • Daily quotations (from BrainyQuote.com).

System Requirements

A Mac with Intel processor and with OS X 10.7 or higher. If you have a PPC (Power PC) processor, OS X 10.4 or earlier or a Windows/Linux machine the application will not work.

Downloading

Go to the Download page.

Installing

Just drag and drop the app into the /Application folder and your done.

Uninstalling

By removing the Jarvis application from the /Application folder.

Removing Jarvis won't remove all the files. So you need to remove the following files and folders :

- Folder com.NightIdeasLab.Jarvis
from
~/Library/Cache
- Folder com.NightIdeasLab.Jarvis.savedState
from
~/Library/Saved Application State
- File com.NightIdeasLab.Jarvis.plist
from
~/Library/Preferences
- File com.NightIdeasLab.Jarvis.plist.lockfile
from
~/Library/Preferences
- File com.NightIdeasLab.Jarvis.LSSharedFileList.plist
from
~/Library/Preferences
- File com.NightIdeasLab.Jarvis.LSSharedFileList.plist.lockfile
from
~/Library/Preferences
